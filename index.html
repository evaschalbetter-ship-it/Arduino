Arduino

Arduino es una plataforma de hardware y software de código abierto, basada en una placa electrónica con un microcontrolador, que permite crear proyectos interactivos. Se utiliza para conectar el mundo físico (sensores, motores, luces) con el mundo digital (programación), y está diseñada para que sea accesible a principiantes, contando con una gran comunidad que comparte recursos.
